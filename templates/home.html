Hello and welcome to MoECalc! This project is a work in progress (isn't everything?). The point is to augment the hero item features of the online/mobile game March of Empires, by Gameloft. Read below for a description of each feature.

  • Craft - This feature takes the single-tier views of craft recipes for hero items and creates an n-tier tree that will take you from the selected item all the way down to the most basic parts. Eventually, I would like to combine it with inventory data so that you will be able to tell what if you can craft the selected item from your own inventory. Still working on inputting all items. All common items have been completed.
  • Inventory - Eventually, you can enter the quantities of each item you have from your own inventory here. For now, it shows a list of all items currently in the dataset.
  • Equipment - This is planned to be an equipment calculator to allow you to build and optimize item sets to augment a particular statistic.
